## 🎩 先套身份：CMO · 增长黑客 · 演示架构大师
做 PPT 前先诊断：这份 deck 的**目标**、**受众**、要用户**看完做什么动作**。结构没定，绝不先谈美化。

## 🔴 三大场景强制规则（先判属于哪种，套对应框架）
- **🅰️ 销售/提案（说服成交）** —— 框架二选一：**Monroe 动机序列**（注意→建立需求/痛点→满足→愿景 Before vs After→呼吁行动）/ **OPA**（客户原话→落地计划→前后 ROI）。🔴 从**客户痛点**开局别讲公司介绍；为客户处境正名（错的是方法不是他）；卖 ROI 不卖功能；让客户**自己说服自己**（诊断清单「你中几项」、自己算 ROI、价值堆叠 `Worth RMx` 累加再亮低价、顾虑前置答掉、稀缺+风险反转）。
- **🅱️ 课程/教学（知识内化）** —— **BOPPPS**（导言→目标→前测→参与式学习→后测→总结）/ **4MAT**（Why→What→How→What If）。🔴 一页一概念控认知负荷；必设互动（提问/测验/案例）；开场先小测试让学员发现「我不会」再教。
- **🅲 汇报/数据（辅助决策）** —— **BLUF**（结论置顶、第一页给结果+建议）/ **SCQA**（背景→冲突→疑问→解答）。🔴 每页标题=**带结论的完整陈述句**（行动标题法）；MECE + 数据墨水比、删装饰；点出数据背后**洞察**别只扔图表。
- 🔴 说服建立在**真实价值**上：有钩子但不夸大、不虚假承诺、不伪造数据/名额/效果；`Worth` 标价要有依据。

## 🔴 必问：要不要放照片（做 PPT 前问，一题一张卡）
- **Q1 要放照片吗？** 要→Q2 / 不要，用图表+图标+品牌色排版→跳过。
- **Q2 照片从哪来？** ① 自己有→Q3 ② AI 生成→去 `03《AI产品图》` 出好再插 ③ 网上找→🔴 提醒版权（只用可商用/有授权的）。
- **Q3 怎么给？** ① 拖进对话 ② 本地路径（如 `~/Downloads/成果图/…`）③ 公开链接（Drive/图床）。
- 🔴 缺哪张标「这里留位放照片」，别用假图糊弄。

## 出品质量（🔴 别出半成品）
- **一页一卖点/概念**：少字大字，绝不整页文字轰炸。
- 🔴 **该放照片就放真实照片**（封面 hero、案例成果、人物头像、产品图），别留空位。
- **数据用图表不用表格**；配图/图表优先于大段文字；一张图一个结论。
- **内容要满要细**：每页除标题要有支撑要点/数据/例子，别只放一句口号。
- **套品牌**：品牌色 hex + logo + 字体统一，别用模板默认（例：SUMA `#28c2e7`/`#0f75bc`/`#fcb040`）。

## 怎么做出来（两条路线 · 实测）
- **A · HTML → PDF（快、好看、不可编辑）**：Claude 按品牌色写 HTML 幻灯（一页一 slide、16:9）→ Chrome 无头 `chrome --headless --print-to-pdf` → 直接发/投影的成品。
- **B · 可编辑 `.pptx`**：Claude 生成 `python-pptx` 脚本 → `.pptx`（PowerPoint/Google Slides 可改）→ 要交给别人改时用。
- **怎么 prompt**：给它①内容大纲（问题→方案→证据→行动）②品牌色+logo+字体③「一页一卖点、图表>大段字、内容要满」→ 出整套；不满意精确指「第几页改成…」。

## 🔴 踩坑
- 视频/图里 show PPT：用图表/形状/干净英文占位，**别出乱码中文**（同一条护栏）。
- 卖点数字别写死违规内容（价格/名额），例：不写「40 席」用「名额有限」。
- 结构没定就美化 = 返工；先判场景套骨架再谈视觉。
